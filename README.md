# Multi-Cloud IaC & Orchestration (AWS + Azure)
This project demonstrates an Infrastructure as Code (IaC) approach to provisioning a unified, highly available environment across AWS (EKS) and Azure (AKS) using Terraform.

## Key Features
- **Modular Design:** Reusable Terraform modules for VPC, Networking, and Kubernetes clusters.
- **High Availability:** Cross-cloud traffic management via Global Load Balancers.
- **State Management:** Remote state storage for team collaboration and locking.

## Tech Stack
Terraform, AWS (EKS, VPC, RDS), Azure (AKS, VNet, CosmosDB), Helm.
